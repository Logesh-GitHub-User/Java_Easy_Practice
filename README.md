### ⭐ Java Practice Questions (Difficulty: Beginner+)

1. **Positive or Negative:**

2. **Sum of Digits:**
   Input a number, add all digits, and print the result.
   Example: `123 → 1+2+3 = 6`

---

3. **Count Characters (Without spaces):**
   Input a sentence and print how many characters it has **excluding spaces**.

---

4. **Print Only Even Numbers from an Array:**
   Input an array (size 5 or 10) and print only the even elements.

---

5. **Find Smallest Number in an Array:**
   Example array: `{4, 9, 2, 7, 1}` → Output: `1`

---

6. **Check Divisibility:**
   Input a number and check if it is divisible by **both 3 and 5**.

---

7. **Print Multiples of 5 up to N:**
   Input a number `n` and print all multiples of 5 up to `n`.

   Input: `"Coding in Java is fun"`
   Output: `5 words`

   ```
   l
   l
   o
   ```
